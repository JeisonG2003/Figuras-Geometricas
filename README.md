# **Proyecto Figuras Geometricas**

Repositorio del proyecto desarrollado para la asignatura **Estructura de Datos (E)** en la **Universidad Estatal Amazónica**, carrera de **Ingeniería en Tecnologías de la Información**.

## **Descripción**

Este proyecto implementa una aplicación en Java para la gestión y cálculo de propiedades de figuras geometricas, permitiendo a los usuarios crear figuras, mostrar su información y calcular áreas y perímetros.
El proyecto utiliza conceptos de POO como herencia, encapsulamiento y polimorfismo, y está organizado en clases: Figura, Rectangulo, Circulo y GestionFiguras (clase principal).
Con este desarrollo se demuestra el uso práctico de los conceptos de la Programación Orientada a Objetos y la estructura de un proyecto Java en NetBeans.

## **Objetivos**

* Aplicar los principios de la POO en la creación de clases y objetos.
* Desarrollar un sistema sencillo para gestionar diferentes figuras geometricas.
* Implementar cálculos de área y perímetro utilizando polimorfismo por sobrecarga de métodos.
* Organizar el código mediante encapsulamiento y herencia para mejorar la mantenibilidad.
* Demostrar la correcta estructura de un proyecto Java ejecutable en NetBeans.
